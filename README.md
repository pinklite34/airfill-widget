# Airfill/Bitrefill Widget

#### Get started

Runs on localhost:8000

```sh
$ git clone git@github.com:bitrefill/airfill-widget.git && cd airfill-widget
$ yarn
$ yarn start
```

#### Run tests

```sh
yarn test:watch
```

#### Build the production bundle

```sh
$ yarn dist
```

#### Run locally in production mode

```sh
$ yarn serve:dist
```
